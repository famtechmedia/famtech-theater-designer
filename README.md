# Fam Tech Media Theater Designer

This is a beginner-friendly React + Vite web app for a Home Theater Designer that you can deploy on Vercel and then embed into your Wix website.

## What this version includes
- Room width, depth, and height
- Screen size
- Projector vs TV
- Dolby Atmos layout selector
- Seating rows and riser
- Live top-view layout
- Download summary button

## How to deploy
1. Upload these files to a GitHub repository
2. Import the repository into Vercel
3. Let Vercel deploy automatically
4. Copy the live Vercel URL
5. In Wix, use Embed Site / iframe and paste the URL
